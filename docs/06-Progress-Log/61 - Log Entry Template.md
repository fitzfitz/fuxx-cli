# 61 - Log Entry Template

Copy the block below into the Entries section of [[60 - Progress Log]] each session.

```
### YYYY-MM-DD — <short title>
- **Did:** <what you actually built/changed>
- **Learned:** <a Rust concept or project insight you now understand>
- **Blocked:** <anything stuck, or "nothing">
- **Next task:** <the single next task, linked to its milestone note>
```
