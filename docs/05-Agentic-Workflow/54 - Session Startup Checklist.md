# 54 - Session Startup Checklist

Run through this at the start of every work session — especially with a fresh AI
assistant that has no memory of past sessions.

## Startup

1. Open [[60 - Progress Log]] and read the **most recent entry**. This is your memory.
2. Note the **current milestone** and its status in [[40 - Development Roadmap]].
3. Identify the **one next task**.
4. If using an AI, paste it the last progress-log entry + the relevant milestone note so
   it's caught up.

## During

5. Use [[53 - Task Handoff Template]] for the task.
6. Follow [[51 - Delegation Principles]] — review, understand, verify.

## Shutdown (do not skip)

7. Update the milestone's status/checklist in its note.
8. Add a new [[60 - Progress Log]] entry: date, what changed, what you learned,
   what's blocked, the next task.
9. Commit to Git (code + vault).
